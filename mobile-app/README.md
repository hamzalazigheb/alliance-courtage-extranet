# Alliance Courtage Mobile App

Application mobile React Native pour iOS et Android.

## Installation

```bash
npm install
# ou
yarn install
```

## Configuration

1. Copier `.env.example` vers `.env`
2. Configurer `API_BASE_URL` avec l'URL de votre API backend

## Développement

### iOS
```bash
npm run ios
```

### Android
```bash
npm run android
```

## Structure du projet

- `src/api/` - Client API pour communiquer avec le backend
- `src/components/` - Composants réutilisables
- `src/screens/` - Écrans de l'application
- `src/navigation/` - Configuration de la navigation
- `src/context/` - Context API pour l'état global
- `src/utils/` - Utilitaires et helpers
- `src/types/` - Types TypeScript
